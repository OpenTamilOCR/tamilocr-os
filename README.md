# TamilOCR OS

> The engineering operating system for the [OpenTamilOCR](https://github.com/OpenTamilOCR) organization.

## What is TamilOCR OS?

TamilOCR OS is the single source of truth for all organizational knowledge — architecture, standards, governance, research, roadmaps, and AI workflows — within the OpenTamilOCR project.

Every repository, every document, every model, every dataset, and every contributor follows the standards defined here.

## Mission

Improve an existing open-source OCR framework to become one of the best Tamil OCR systems available.
Version 1 targets printed Tamil documents.

## Getting Started

**→ [SYS-000 Master Index](SYS-000_Master_Index.md)** — the root catalog of this knowledge base. Start here.

The Master Index defines the complete document hierarchy, the canonical ID system, the 8-tier knowledge architecture, the governance pipeline, and all conventions required to read, write, or extend this knowledge base.
